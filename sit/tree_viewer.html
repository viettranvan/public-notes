Glossary — tên trên UI ↔ loại org unit LMS
  • Cluster ↔ LMS School
  • Programme ↔ LMS Program
  • Module ↔ LMS Module (định nghĩa môn, giữ qua các trimester)
  • Course / Offering ↔ LMS Course Offering (nơi có roster / enrollment)
  • Semester ↔ LMS Semester (mã 4 số = theo kỳ; khác 4 số = perpetual candidate)
